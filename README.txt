KLAAR PAKKET

Vervang in je project root gewoon:
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png
- maskable-512.png
- favicon-32.png

Daarna:
1. push naar GitHub
2. wacht op Vercel deploy
3. verwijder oude DriveNav snelkoppeling van Android
4. open je live site in Chrome
5. installeer opnieuw

Dit pakket bevat:
- jouw gekozen logo
- werkende Android install knop
- echte PWA setup
- telefoonnummer optioneel
- bel/WhatsApp alleen zichtbaar als telefoonnummer bestaat
