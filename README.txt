- manifest.webmanifest

(zie head-snippet.txt)

Belangrijk op Android:
1. Verwijder oude DriveNav-snelkoppelingen van je beginscherm.
2. Push de nieuwe bestanden naar GitHub.
3. Laat Vercel opnieuw deployen.
4. Open de live site in Chrome.
5. Kies 'App installeren' of 'Toevoegen aan startscherm'.

Belangrijk op iPhone:
1. Open de live site in Safari.
2. Tik op Deel.
3. Kies 'Zet op beginscherm'.

Als je nog steeds het Chrome-logo ziet, dan is het meestal cache:
- verwijder de oude snelkoppeling
- sluit Chrome/Safari volledig af
- open de live URL opnieuw
- installeer opnieuw
